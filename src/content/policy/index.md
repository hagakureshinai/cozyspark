# このサイトについて｜サイトポリシー

## 1. 著作権について

当サイトに掲載されている小説・文章等のオリジナルコンテンツの著作権は、Cozy Spark運営者に帰属します。

一方、使用している一部の画像・アイコン素材等の著作権は、各権利者に帰属します。

## 2. 免責事項

当サイトに掲載しているすべての物語はフィクションであり、実在の人物、団体、場所とは一切関係ありません。

当サイトの利用により生じた損害等について、運営者は一切の責任を負いません。

## 3. 年齢制限について

R18作品の取り扱いについては、外部サービスへのリンクで対応しています。

当サイト内には成人向け表現を想起させる内容が含まれるため、18歳未満の方の閲覧はご遠慮ください。

## 4. アクセス解析ツールについて

当サイトでは、サイトの利用状況を把握し、サービス向上のためにGoogle Analyticsなどのアクセス解析ツールを使用する可能性があります。

これらのツールは、トラフィックデータの収集のためにCookieを使用しますが、個人を特定するものではありません。
